# Director AI Character Bible v1.0

## 1. Purpose

This document defines the standard for creating,
maintaining, and controlling character identity
inside Director AI.

The goal:

- Keep characters consistent across episodes
- Maintain visual identity
- Maintain personality continuity
- Support long-running AI storytelling


# 2. Character Identity Rules

Every character MUST have:

- Unique name
- Role in story
- Age
- Appearance
- Personality
- Background
- Voice style
- Visual reference

# 4. Visual Consistency Rules


Every generated scene MUST maintain:


## Face

DO:
- Same facial structure
- Same identity

DO NOT:
- Change face
- Change age randomly


## Hair

DO:
- Maintain hairstyle

DO NOT:
- Change hairstyle without story reason


## Clothing

DO:
- Keep main costume

Allowed:
- Upgrade costume
- Damage during battle
- Change for special events


# 5. Character Evolution Rules


Characters can evolve:


Allowed:

- New skills
- Emotional development
- New equipment
- Costume improvement


Not allowed:

- Different person
- Different identity
- Random appearance change
